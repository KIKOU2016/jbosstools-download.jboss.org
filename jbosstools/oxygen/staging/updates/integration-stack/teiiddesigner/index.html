JBoss Tools Teiid Designer Staging Update Site

Latest Build: 11.1.0.Final

This is the Staging Release Update Site for Teiid Designer.

    Teiid Designer is available through the Integration Stack or directly through this update site. For Integration Stack installation instructions please see JBoss Tools Integration Stack community installation.

  1. To install directly from this site, start Eclipse native or JBoss Tools Eclipse.

  2. Go to Help > Install New Software... > Add...

  3. Paste in the URL of this update site. Select 'Data Virtualization' from the list of features and restart when prompted.
